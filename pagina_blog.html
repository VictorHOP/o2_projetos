GitHub Sponsors: apoio financeiro para desenvolvedores chega ao Brasil

01/06/2022

Dashboard

GitHub Sponsors já está disponível aos usuários brasileiros, basta criar um perfil na plataforma e cadastrar uma conta bancária do Brasil.
A funcionalidade é importante para os desenvolvedores e organizações que desejam receber apoio financeiro dos seus usuários, clientes ou fãs. Basta possuir uma conta bancária para doar uma quantia de dinheiro no GitHub Sponsors.

"Fazer parte do Sponsors realmente aumentou a visibilidade tanto do meu perfil quanto do perfil da organização na qual mantenho um projeto de código aberto. É muito gratificante ver o apoio e o reconhecimento de seu trabalho por parte da própria comunidade", disse a engenheira de back-end Camila Maia.

A comunidade do GitHub tem crescido constantemente ao redor do mundo e aqui no Brasil não tem sido diferente. Nos últimos anos, o crescimento em território nacional foi de 40% e, hoje, a plataforma já abriga 2 milhões de desenvolvedores — entre eles, milhares ajudam a comunidade do open source.

Na próxima quinta-feira (2), o GitHub realizará um encontro virtual em uma transmissão ao vivo no YouTube, às 18 horas, para discutir sobre manutenção do open source e oferecer dicas sobre o Sponsors aos desenvolvedores.
