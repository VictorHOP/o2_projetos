GitHub Sponsors: apoio financeiro para desenvolvedores chega ao Brasil

01/06/2022 | Tecnologia

Nesta quarta-feira (01/06/2022), a plataforma GitHub anunciou um novo recurso para ajudar os desenvolvedores que usam o site e criam ou mantém projetos open source. A funcionalidade GitHub Sponsors já está disponível aos usuários brasileiros, basta criar um perfil na plataforma e cadastrar uma conta bancária do Brasil.
